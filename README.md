#  Gerenciador de Tarefas com Data e Hora

Um projeto simples em **Python** que usa o módulo `datetime` para registrar **a data e hora em que cada tarefa é criada**.  
Ideal para iniciantes que desejam praticar manipulação de datas e persistência de dados.

---

##  Funcionalidades
- Adicionar tarefa com data/hora automáticas  
- Listar tarefas  
- Marcar como concluída  
- Salvar e carregar do arquivo `.txt`

---

##  Tecnologias usadas
- Python 3+
- Módulo `datetime` (biblioteca padrão)
